# 3D snake for browser using Three.js 
## Controls:
- **ENTER** - ___Start game___
- **W** - ___Go forward___
- **S** - ___Go back___
- **A** - ___Go left___
- **D** - ___Go right___
- **R** - ___Go up___
- **F** - ___Go down___

### Pay attention to the buttons on the top right!
***they will help you customize the view***
